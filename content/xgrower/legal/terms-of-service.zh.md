# X Grower 服务条款

本服务条款（以下简称"本条款"）适用于您对 X Grower（以下简称"本服务"）的访问和使用。本服务包括一款 Chrome 浏览器扩展程序及其配套网站，由 GrowthHunt（以下简称"我们"）运营。您在安装扩展、注册账户或以任何方式使用本服务时，即视为您已阅读、理解并同意本条款。如不同意，请勿使用本服务。

## 1. 服务说明

### 1.1 X Grower 的功能
X Grower 是面向 x.com（原 Twitter）平台用户的效率工具。本服务通过 AI 生成回复草稿，并在您的指令下，于您本机已登录的浏览器会话中执行发送动作。AI 草稿的生成由我们的后端调用第三方大语言模型完成，具体的发送行为发生在您本地的 Chrome 浏览器中。

### 1.2 与 X 平台的关系
我们是独立的第三方工具提供方，与 X Corp.、Twitter Inc. 及其关联公司之间不存在任何附属、授权、合作或代理关系。产品名称及说明中提及"X"仅为客观描述。

### 1.3 不接触您的 X 账号凭证
我们不会请求、收集、存储或传输您的 X 用户名、密码、登录 Cookie、两步验证代码或会话令牌。扩展仅在您本人已经登录的页面中运行，并通过您自身的浏览器会话提交内容。

## 2. 用户资格与账户

### 2.1 年龄与行为能力
您应当年满 18 周岁，且具有签订合同所需的完全民事行为能力。

### 2.2 注册
使用付费功能需在 xgrower.growthhunt.ai 通过 Google OAuth 或邮箱魔法链接完成注册。您应妥善保管账户登录信息，并对账户下发生的一切行为承担责任。

### 2.3 单一账户原则
每位自然人仅可注册一个 X Grower 账户。通过注册多账户绕开速率限制、试用额度或退款政策的，构成违约，我们有权终止账户。

账户处于正常状态时，积分不设过期。如账户因违规被终止，剩余积分作废，不予退还。
