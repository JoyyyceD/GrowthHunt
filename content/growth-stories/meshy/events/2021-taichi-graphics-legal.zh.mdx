<Catalyst>
大多数 AI 应用层创始人是先创业，再去建底盘。胡渊鸣是先建底盘——八年学术图形学的 Taichi Lang 工作——然后悄悄把它做成一家产品公司。这门 27K 星的 GPU 语言不是副业，它就是产品化的文本到 3D 流水线真正运行其上的发动机。
</Catalyst>

2021 年 8 月。胡渊鸣（Ethan Hu）在 MIT EECS / CSAIL 完成博士答辩，导师是 Frédo Durand 与 Bill Freeman——计算机图形学领域最具影响力的两位学者。论文题目是 *The Taichi High-Performance and Differentiable Programming Language for Sparse and Quantized Visual Computing*。

论文本身评价很高，SIGGRAPH 2022 评出杰出博士论文荣誉提名。但比这份文档更重要的，是与论文一同存在的那个工件：**到 2021 年时，开源 Taichi 编程语言已经在 GitHub 上有 25K+ 星标，被 300+ 家机构用于物理仿真和图形学研究。**

## Taichi 是什么

Taichi 是一门面向稀疏与量化视觉计算的可微分、高性能 GPU 编程语言。换句话说：它是介于 Python 级别科学代码与 GPU 原生执行之间的一层；让研究者写物理仿真、流体力学、稀疏神经场、以及（后来的）3D 生成模型时，工程量只占纯 CUDA 栈的一个零头。

| Taichi 指标 | 数值（2021.8） |
|---|---|
| GitHub 星标 | 约 25,000 |
| 采用机构 | 300+ |
| 首篇 ACM TOG 论文 | 2019 年 12 月 |
| 学术引用 | 数千 |
| 已存在法律实体 | Taichi.graphics（2021 年起） |

等到 18 个月后一家叫 Meshy 的 3D 生成公司出现时，这块底盘已经深到——**3D / 图形学社区里每一位有技术信用的评估者，对这位创始人都默认带着正面预期。**

## 底盘形态 (d)，图形学版本

GTM 互索把「创始人声誉作为底盘」归为 A1 形态 (d)。Linear 的 Karri Saarinen 是前 Coinbase + Sense 的设计声誉。Anthropic 创始人是 OpenAI 的扩展工程声誉。Hugging Face 是学术 NLP 社区声誉。

Taichi 是第四种形态：**学术图形学基础设施底盘。** 它更像 Hugging Face 的模式（社区引力型开源信用），不像 Linear（商业产品声誉）。差异点在于：Taichi 与产品化的产品之间，技术上是「相邻」的。可微分仿真直接映射到「3D 网格上的扩散模型」问题——而这种「底盘 → 产品」的直接映射，在创始人底盘里非常稀有。

> 一位「套壳层」创始人 2023 年宣称要做好 3D-AI，需要凭空制造可信度。一位 Taichi 作者 2023 年宣称要做好 3D-AI，可信度已经写在底盘里。买方不需要再评估创始人——GitHub 历史已经评估过了。

## 这块底盘买到了什么，又买不到什么

底盘买到三样东西：

- **进入争夺品类的入场券。** 2023 年的 3D 生成赛道里 Tripo、Luma、Rodin、CSM、以及若干学术 Stable3D 分叉同时挤在六个月窗口里。底盘决定了哪些项目优先获得机构关注。
- **招聘漏斗。** Taichi Lang 社区的贡献者，以及清华 / MIT 图形学相邻的研究者，是天然的候选池。Meshy 不少研究 / 工程岗位的人，来自胡渊鸣 Meshy 之前的 Taichi 网络。
- **长内容访谈杠杆。** 当胡渊鸣后来上播客（XR AI Spotlight、WhynotTV）时，对话可以一直停留在技术深度上而不流失观众。「创始人即 IP」一般会有 18 个月冷启动窗口的问题——但当底盘已经存在时，这个窗口不适用。

底盘买不到的：

- **媒体注意力。** 没有 TechCrunch 报道 Taichi 这家公司。没有 Bloomberg 报道 Meshy A 轮。底盘是社区引力型，不是媒体引力型。
- **消费品牌。** 在游戏开发 / 图形学 / 仿真这些圈子之外，Taichi 没多少人知道。Meshy 在 3D 打印或电商社区做品牌，仍然要从零起步。
- **规模化阶段的资本效率。** 底盘换来第一张支票。从 1,500 万 ARR 到 4,000 万 ARR 这条路，仍然要靠 PRNewswire 与 36Kr 来推。

<Conclusion title="可以复用的部分">

- **如果可以，先把底盘建起来。** 八年公开的开源工作，是 2026 年 AI 赛道里最稀有的创始人资产。如果你拥有它，进入品类的边际成本将会是数量级地降低。
- **挑一块在技术上与最终产品相邻的底盘。** Taichi → 3D-AI 是直接映射。不能映射到产品的底盘（一个营销小工具）不会换来底盘信用。
- **不要指望底盘在它原生社区之外做品牌的事。** 底盘买到的是入场券，市场份额仍然要靠产品节奏与桥接集成去赢。

</Conclusion>
