<Catalyst>
距最初 Note 众筹整整两年，Plaud 需要一款旗舰产品。Note Pro 不是替代品——它是一个三层产品线的新顶端，让 Plaud 在不放弃 159 美元入门价格点的前提下，向上销售。
</Catalyst>

2025 年 8 月 27 日，Plaud 开启 Plaud Note Pro 预购，售价 179 美元。同款磁吸卡片外形，内部大幅升级。

| 参数 | Plaud Note（2023 年） | Plaud Note Pro（2025 年） |
|---|---|---|
| 麦克风 | 2 个 MEMS + 1 个 VPU | 4 个 MEMS + 1 个 VPU |
| 拾音范围 | 9.84 英尺 | 16.4 英尺（50 小时 @ 9.8 英尺） |
| 屏幕 | 无 | 0.95 英寸 AMOLED |
| 存储 | 64GB | 64GB |
| 录音容量 | 30 小时 | 标准 30 小时 / 延长模式 50 小时 |
| 机载 AI | 无 | 「按键高亮」 |
| 发布价格 | 159 美元 | 179 美元 |

同一个媒体周期还带来了 Plaud Intelligence 3.0——一次重大的 App 和 Web 端更新，新增多模态输入（音频 + 文本 + 图像 + 高亮标记）、360° 视图角色化摘要，以及跨 OpenAI、Anthropic 和 Google 模型的动态路由。**硬件发布与软件平台发布在同一个媒体窗口落地。**

## Note Pro 如何改变了产品线格局

原版 Note 是 Plaud 磁吸卡片形态下唯一的 SKU，独自撑了 26 个月。到 2025 年 8 月，这款 SKU 正在独自承担品牌——在 Plaud 本应主导的品类里。

Note Pro 的推出重塑了产品线：

- **Plaud Note（159 美元）**——入门款，双麦克风，无屏幕。摆上 Best Buy / Costco 货架的那款。
- **Plaud Note Pro（179 美元）**——旗舰款，四麦克风，AMOLED 屏，专业级可靠性。
- **Plaud NotePin（169 美元）**——可穿戴，解放双手，面向不同买家。

Note 和 Note Pro 之间 20 美元的价差是刻意为之的。卖点不是"买这个代替那个"，而是"如果你是专业用户——记者、律师、医生、高管——就买这款"。**大多数 Note Pro 买家不是从 Note 升级上来的；而是在购买决策的那一刻，被引导到了更高的产品层级。**

这是一个在硬件领域执行的经典价格锚定动作。Pro 层级提升了标准款的感知价值；标准款保护了销量基本盘；可穿戴款增加了第三个维度。整个产品线在不蚕食现有渠道的前提下，扩大了目标买家的总量。

## "实时人机对齐"——为什么这个定位说法很重要

Plaud 的发布新闻稿将 Note Pro 称为"全球首款支持实时人机对齐的 AI 记录仪"。主打功能："按键高亮"——设备上的一个实体按钮，让用户能够实时向 AI 标记录音中哪些时刻最重要。

这个营销语言背后，是一个具体的产品决策。之前的 Plaud SKU 录下所有内容，让 AI 猜什么重要。Note Pro 让用户实时引导 AI。返回的摘要，会偏向用户标记的时刻，而不是 AI 认为显著的时刻。

为什么这不只是一个功能：

- **这是科技评测人能讲述的故事。** "实时人机对齐"是一个品类主张，不是规格表上的一行参数。Brian Heater 在 2025 年 12 月 TechCrunch 年终回顾中把设备标题定为"一款我随身携带的出色 AI 录音机"——而他评测里的主打功能，正是这个。
- **这是能支撑一个价格层级的功能。** 四麦克风是渐进式升级，AMOLED 屏是渐进式升级，实时人机对齐是一个跨品类的主张。整个组合让这款设备定价 179 美元，而不是带屏幕的 159 美元。
- **这为 Intelligence 3.0 的定位铺路。** 软件现在被描述为"硬件的操作系统"——而不是一款转录 App。Plaud Intelligence 由此成为科技评测人、企业买家和平台合作伙伴可以在另一个层次与之互动的产品层，而不只是"Plaud 的 App"。

## 与 Plaud Intelligence 3.0 的捆绑发布

Plaud Intelligence 3.0 与 Note Pro 同日发布。平台发布新增了三个功能表面：

- **多模态输入。** App 支持音频、文本、图像和高亮标记的组合输入——比转录 App 更全面的输入表面。
- **360° 视图摘要。** 基于角色的模板，根据你的身份（医生、律师、记者、高管）改变摘要格式。超过 10,000 个模板的库，是对抗通用转录 App 的护城河。
- **动态 LLM 路由。** 平台根据任务类型，在 OpenAI、Anthropic 和 Google 模型之间动态分发请求。Plaud 不被任何一家 LLM 供应商锁定。

硬软件捆绑完成了单独硬件发布无法完成的工作。Note Pro 是一款 179 美元的更好录音机；Intelligence 3.0 是一个真正的软件平台。两者合在一起，才能支撑 2025 年 10 月开发者平台发布时将要使用的"平台"定位框架。

> Plaud 在 2025 年 8 月这次发布里，交付的不是一款新产品，而是一个新的定位。硬件是证明点；软件是主张。

## 50% Pro 计划转化率作为单位经济模型的锚点

Plaud 在 Note Pro 发布窗口披露，约 50% 的 Plaud 买家已升级为付费 Pro 计划（99.99 美元/年）或 Unlimited 计划（239.99 美元/年）。以 Sacra 在 Note Pro 发布前的估算（1.8 亿美元年化营收）和约 70 万台出货量来推算，订阅线已是总营收中可观的一部分。

Note Pro 的发布通过两种方式强化了转化机制：

- **预购赠品：600 分钟免费转录时长。** 相当于两个月的 Pro 计划，随设备附赠。等免费时长用完，用户已经进入工作流程。
- **将设备定位为专业级。** Note Pro 买家比 Note 买家更可能是每日高频使用的专业用户。每日使用会更快耗尽免费入门版的时长上限，加速付费转化。

Plaud 在同一窗口报告，**Note Pro 预购买家中，超过 50% 在结账时同步订阅了 Pro 计划**——高于基础转化率。旗舰层级的转化率更高，而这正是价格锚定动作所设计要产生的结果。

<Conclusion title="可以复用的东西">

- **设一个价格溢价小、规格溢价也不大的旗舰层级。** Note Pro 比 Note 贵 20 美元是刻意为之。溢价太大会失去价格锚定效果；溢价太小会模糊层级界限。正确的溢价，是那个让旗舰款感觉像"同款产品，为认真用户而生"的溢价。
- **硬件发布与软件平台发布捆绑在一起。** 单独的 Note Pro 只是规格升级。Note Pro + Intelligence 3.0 是一个品类主张。组合后的表面，给评测人和分析师提供了一个单纯硬件刷新讲不了的故事。
- **用一个功能主张锚定媒体周期。** "实时人机对齐"是每篇 Note Pro 评测都借用的那句话。选一个你的买家会反复提及的动词式功能，让规格表的其余部分成为佐证。

</Conclusion>
